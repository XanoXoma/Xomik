# Xomik
В данной программе наглядно показывается физический эффект домино - это физическое явление, при котором падение одного объекта запускает цепную реакцию последовательного падения других объектов. Название происходит от игры в домино, где костяшки выстраивают в ряд, и падение первой вызывает падение всех остальных. 
В программе можно менять колличество домино их параметры (высота, ширина, длина) а также цвет и интервал растоновки домино, так же можно менять силуи и направление толчка домино, все это изменяется в обекте "EffectDomino"
Вся програма построена на скрипте привязаному к обекту "EffectDomino" при запуске создаются доп. объекты и над ними производятся действия также скрипт добавляет физику обектам а именно  Rigidbody для падения и AddForce для толчка домино.
